# Paper References

This folder stores bibliography files, paper notes, and reading summaries for the project.

## Suggested Reading Tracks

1. Medical imaging datasets: BraTS, MIMIC-CXR, CheXpert, CAMELYON, PANDA, FastMRI.
2. Vision encoders: ViT, Swin Transformer, ConvNeXt, 3D Swin.
3. Biomedical text encoders: BioBERT, ClinicalBERT, PubMedBERT.
4. Cross-modal alignment: CLIP-style contrastive learning and InfoNCE.
5. Retrieval: FAISS, approximate nearest-neighbor search, case-based reasoning.
6. Quantization: product quantization, vector quantization, compressed retrieval.
7. Explainability: Grad-CAM, attention rollout, retrieval-based explanations.

## Notes

Use `references.bib` as the shared BibTeX source for the LaTeX manuscript. Verify venue metadata and page numbers before submission.
